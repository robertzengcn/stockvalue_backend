简单来说，如果把投资比作“相亲”，**M-Score 是用来检测对方有没有“整容/虚报身高”（财务造假）**，而 **F-Score 是用来评估对方的“身体素质和家底”（经营质量）**。

在 A 股和港股这种财报透明度差异较大的市场，这两个分数是价值投资者的“照妖镜”。

## ---

**1\. Beneish M-Score：财务造假探测器**

由 Messod Beneish 教授开发，它是一个概率模型，通过 8 个财务指标的变动来判断一家公司是否**操纵了利润**。

### **核心公式**

M-Score 是一个综合得分，公式如下（别被吓到，AI 会帮你算）：

$$M \= \-4.84 \+ 0.92 \\times DSRI \+ 0.528 \\times GMI \+ 0.404 \\times AQI \+ 0.892 \\times SGI \+ 0.115 \\times DEPI \- 0.172 \\times SGAI \+ 4.679 \\times TATA \- 0.327 \\times LVGI$$

### **关键看点：**

* **DSRI (应收账款指数)：** 如果应收账款增长远超营收增长，可能是在虚增收入。  
* **GMI (毛利率指数)：** 毛利率异常下滑，公司更有动机粉饰报表。  
* **TATA (应计总额/总资产)：** 利润中如果没有现金流支撑（全是会计数字），造假概率极大。

**📌 判别标准：**

* **$M \> \-1.78$：** 高风险！公司很有可能存在财务操纵。  
* **$M \< \-1.78$：** 相对安全，操纵概率较低。

## ---

**2\. Piotroski F-Score：公司质量评分**

由芝加哥大学 Joseph Piotroski 教授提出。它不看估值便宜不便宜，只看公司的**基本面是否在好转**。

### **评分维度 (共 9 项，每达标一项得 1 分)**

F-Score 将公司分为三个维度进行“体检”：

| 维度 | 指标 (得 1 分的条件) | 逻辑 |
| :---- | :---- | :---- |
| **盈利能力** | 资产收益率 (ROA) \> 0 | 公司在赚钱 |
|  | 经营现金流 (CFO) \> 0 | 赚的是真钱，不是白条 |
|  | 今年 ROA \> 去年 ROA | 盈利能力在提升 |
|  | **CFO \> ROA** | **核心！** 现金流比净利润还健康（防造假） |
| **杠杆与流动性** | 长期破产风险下降 | 负债率比去年低 |
|  | 流动比率提升 | 短期还钱能力增强 |
|  | 未增发新股 | 没有通过摊薄股东权益来圈钱 |
| **运营效率** | 毛利率提升 | 产品竞争力变强 |
|  | 资产周转率提升 | 卖货变快了，效率提高 |

**📌 判别标准：**

* **8 \- 9 分：** 极优。基本面非常扎实且在改善。  
* **0 \- 2 分：** 极差。基本面正在迅速恶化，即便 PE 再低也要小心。

## ---

**3\. 两者结合的“实战威力”**

在你的 AI 软件中，这两个指标的组合策略非常强大：

1. **第一步（M-Score）：** 排除掉 $M \> \-1.78$ 的公司。不管它业绩多好，只要有造假嫌疑，直接剔除（在 A 股这能帮你避开 80% 的雷）。  
2. **第二步（F-Score）：** 在剩下的公司里找 $F \\ge 7$ 的公司。这些是真正业务向上、现金流稳健的优等生。  
3. **第三步（你的利差模型）：** 看看这些“不造假且身体好”的公司，分红是不是比银行存款高。
